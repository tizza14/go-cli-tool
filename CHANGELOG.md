## [1.0.0] - 2024-01-19

### Added
- Initial release of Go CLI Tool
- Root command with version and help flags
- `hello` command for greeting with customizable name and uppercase option
- `version` command for displaying version information
- Configuration support via Viper (file, environment variables, and flags)
- Comprehensive unit tests for all commands and internal packages
- Integration tests for end-to-end testing
- Makefile for common development tasks
- GitHub Actions CI/CD workflows
- Complete documentation (README, CONTRIBUTING, this CHANGELOG)
- golangci-lint configuration for code quality
- MIT License
